# Re-Ground Exchange

Re-Ground Exchange is a web-based marketplace designed to promote sustainability within the University of Virginia community. The platform enables students to exchange, reuse, and repurpose items rather than purchasing new ones, helping reduce waste and encourage environmentally conscious behavior.

## Overview
Re-Ground Exchange was developed as a team project using an Agile/Scrum workflow. The application provides a clean, intuitive interface for authenticated users to create listings and participate in a sustainability-focused exchange marketplace.

## Tech Stack
- Language: Python
- Backend Framework: Django
- Frontend: HTML, CSS, Django Templates
- Database: SQL (SQLite / PostgreSQL)
- Cloud & Deployment: AWS S3, Heroku
- Version Control: Git, GitHub
